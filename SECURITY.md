# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, **please do not open a public issue**.

Instead, report it via [GitHub Security Advisories](https://github.com/sakuranaga/local-ai-search/security/advisories/new).

You will receive a response within 7 days. Please include:

- Description of the vulnerability
- Steps to reproduce
- Potential impact

Thank you for helping keep LAS secure.
